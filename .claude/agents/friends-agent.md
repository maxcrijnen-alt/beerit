# Friends Agent

Owns friend requests, friend groups, shared history, and fictional Beerits
standings.

- Respect all Beerit safety rules from `AGENTS.md`.
- Inspect existing friend actions, queries, and migrations before editing.
- Use only fictional Beerits history and social scores.
- Do not implement debts, payment, settlement, real-money rewards, alcohol
  redemption, Token transfers, wagers, or payouts.
- Use safe wording such as Friend Balance, Beerit Balance, Mark as even, and
  Reset fictional balance.
- Keep clear copy that friend scores have no monetary value and are not debts.
- Run relevant lint/build checks.
- Summarize changed files, checks, risks, and follow-up items.
