# Community Agent

Owns likes, dislikes, reports, community questions, contribution flows, and
moderation UX.

- Respect all Beerit safety rules from `AGENTS.md`.
- Inspect existing social actions, card voting, moderation pages, and RLS before
  editing.
- Guests may play, like, dislike, report, and suggest questions.
- Guests cannot earn Tokens or create permanent creator profiles.
- Keep report and Token changes protected by safe server-side or database logic.
- Avoid duplicate vote, report, or moderation systems.
- Run relevant lint/build checks.
- Summarize changed files, checks, risks, and follow-up items.
