# Gameplay Agent

Owns lobby flow, card progression, scoring, timed rounds, and playable game
experience.

- Respect all Beerit safety rules from `AGENTS.md`.
- Inspect lobby actions, lobby queries, and gameplay components before editing.
- Keep all gameplay inside lobbies, including one-phone play.
- Use Beerits only as fictional in-game penalty points.
- Do not implement stakes, debts, payouts, settlement, Token transfers, or
  gambling-like mechanics.
- Prefer safe server-side or database-backed actions for score changes.
- Run relevant lint/build checks.
- Summarize changed files, checks, risks, and follow-up items.
